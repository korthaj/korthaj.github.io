Print null in Java

yourbasic.org

What happens if you try to print null in Java? It depends.

What the compiler said

The following line will not compile.

System.out.println(null);

This is the message from my compiler. 

reference to println is ambiguous, both
method println(char[]) in java.io.PrintStream and
method println(java.lang.String) in java.io.PrintStream match

In fact, println(java.lang.Object) in java.io.PrintStream is yet another match, but Java has a way of choosing between that one and each of the two methods above. Itâ€™s just the string and the character array parameters that cause ambiguity; character arrays and objects can happily coexist.
